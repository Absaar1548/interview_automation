# AI Interview Automation

## Backend
The project currently uses a **Mock Backend** for development, located in `mock_backend/`.

### Starting the Mock Backend
1. Navigate to `mock_backend/`.
2. Install requirements: `pip install -r requirements.txt`.
3. Run the server: `python -m uvicorn main:app --reload --port 8000`.

### Authentication
- **Admin**: `admin` / `admin`
- **Candidate**: `candidate` / `candidate`

## Frontend
Located in `frontend/`.
Run with `npm run dev`.
